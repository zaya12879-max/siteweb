## The 3-Step Deployment

### Step 1: Create Repository
1. Go to [github.com/new](https://github.com/new)
2. Name it `adsdo-agency`
3. Make it **Public**
4. Click **Create repository**

### Step 2: Push Code
```bash
git init
git add .
git commit -m "Initial commit: ADSDO Agency landing page"
git remote add origin https://github.com/USERNAME/adsdo-agency.git
git branch -M main
git push -u origin main
```

### Step 3: Enable GitHub Pages
1. Go to **Settings** → **Pages**
2. Select branch: **gh-pages**
3. Click **Save**

**Your site is live in 2-5 minutes at: https://username.github.io/adsdo-agency/**
